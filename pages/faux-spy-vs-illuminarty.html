The one difference that changes everything

On paper, Faux Spy and Illuminarty look similar: both detect AI-generated images, both identify which generator made them, both offer free and paid tiers at comparable prices. The numbers are close.

The workflow is not. Illuminarty requires you to download the image and upload it to their site. That's a three-step process: right-click → Save image → Navigate to illuminarty.ai → Upload. When you're on a dating app and something looks off about a profile photo, you're not going to do that five times across a profile. You'll skip it.

Faux Spy is one step: hover and click. The result appears as an overlay on the same page. You never leave where you are. That's not a small UX difference — it's the difference between using the tool consistently and not using it at all.

No competitor has a browser extension. Faux Spy is the only AI image detector that works on any website in Chrome without requiring you to download or upload anything.

Accuracy: both are top-tier, Faux Spy has the edge

Illuminarty's detection accuracy has been independently tested at approximately 91% across major generators. That's legitimately good — it's in the top tier of consumer AI detection tools.

Faux Spy is powered by Hive Moderation's detection engine, which benchmarks at 94% accuracy across Midjourney, DALL-E, Stable Diffusion, Flux, and other major generators. Hive is an enterprise-grade API used by media companies and social platforms. Faux Spy brings that same engine into a consumer browser extension.

In practice, a 3-percentage-point accuracy difference rarely matters for individual checks. What matters is whether you actually run the check. The tool you use is more accurate than the tool you don't.

Who might prefer Illuminarty

Illuminarty makes sense if you need the heatmap overlay. That feature shows exactly which areas of an image have the strongest AI-generation signals — useful if you're doing forensic analysis, writing a report, or need to demonstrate to someone else where the tell-tale signs are.

It also makes sense if you're on a non-Chrome browser (Illuminarty is browser-agnostic since it's a website), or if you're checking images from files on your computer rather than images on websites.
