How each tool works

FaceCheck.ID searches a database of faces indexed from across the internet. Upload a photo and it runs facial recognition to find where that face has appeared — other social media profiles, news photos, websites. With nearly 5 million monthly visitors, it's one of the most widely used face search tools available to individuals.

Faux Spy doesn't search the internet. It analyzes the image directly — the mathematical signatures that AI generators embed in pixel data. When Midjourney or DALL-E creates a face, it leaves statistical patterns that don't appear in genuine photographs. Faux Spy detects those patterns and returns a verdict.

The key difference: FaceCheck needs an existing online history to find. Faux Spy needs an image to analyze. An AI-generated face has no history — FaceCheck finds nothing, Faux Spy finds the AI signature.

The recommended workflow: Faux Spy first, FaceCheck second

These tools are more valuable together than either one alone. Here's the most efficient approach for verifying a profile photo:

  1. Run Faux Spy first. Right-click the profile photo in Chrome and click Investigate. This takes three seconds and costs nothing. If it returns AI Photo with high confidence, you're done — the face is generated and FaceCheck will find nothing useful.
  2. If Faux Spy returns Real Photo, run FaceCheck.ID. Upload the image to facecheck.id to check if the face appears elsewhere online under a different name. This catches the stolen-photo attack vector that Faux Spy doesn't cover.
  3. Treat results as signals, not verdicts. No detection tool is 100% accurate. High AI confidence is a strong red flag. A clean result on both tools is reassuring but not a guarantee.

FaceCheck charges per search — starting Faux Spy first lets you skip the paid search entirely when a photo is AI-generated, which is increasingly common.

Common questions

What's the difference between FaceCheck.ID and Faux Spy?

FaceCheck.ID finds where a face appears online (useful for catching stolen photos). Faux Spy detects whether a photo was generated by AI (useful for catching AI-generated profiles). They solve different problems and work best together.

Does FaceCheck.ID detect AI-generated photos?

No. FaceCheck searches for a face in its indexed database of online images. An AI-generated face has never appeared online — there's nothing to find. Faux Spy analyzes the image itself for AI artifacts, which is why it catches what face search misses.

Is FaceCheck.ID free?

FaceCheck.ID uses credits, approximately $2.99–$9.99 per search. Faux Spy is free for 10 checks per day with no account. Starting with Faux Spy lets you skip the FaceCheck cost whenever a photo turns out to be AI-generated.

Which tool should I use to catch a catfish?

Both — for different attack vectors. Use Faux Spy to check for AI generation. If the photo appears real, use FaceCheck.ID to check if it's a stolen photo. AI-generated faces are increasingly common on dating apps; catching them requires AI detection, not face search.
