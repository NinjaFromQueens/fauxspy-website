Frequently asked questions

Is this really free?+

Yes. The free tier gives you 10 image scans per day — no account, no email, no signup. Pro ($9.99/month) unlocks unlimited scans, AI Photo vs. AI Art distinction, and deepfake detection.

Do I need to install an extension?+

No — this page works in any browser with no extension required. If you want right-click detection directly on images as you browse (no uploading), you can install the Chrome extension or Firefox add-on.

How accurate is it?+

Approximately 94% on images from major generators. The confidence score matters: 90%+ is very reliable, 60–80% is ambiguous and should be treated as one signal alongside other checks. No detector is 100% accurate against the latest models.

Is my image stored or shared?+

No. For URL scans, only the URL is passed to the detection API — we never download or store the image. For file uploads, the image is held in memory during the scan and discarded immediately. Results may be cached for up to 30 days against the same URL for performance.

Why does the URL check sometimes fail for social media images?+

Instagram, X/Twitter, Facebook, and TikTok block cross-origin image fetches. If a URL from one of those sites fails, save the image to your device (right-click → Save image) and upload it using the Upload tab instead — that path works for any image you can save locally.
